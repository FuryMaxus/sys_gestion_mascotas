# Sanos y Salvos - MS Mascotas

Este microservicio utiliza Gradle para aprovechar la compilación incremental y agilizar el desarrollo bajo la estrategia Trunk-Based Development.

## Requisitos previos
- Java 21

## Instalación y Ejecución
1. Abre una terminal en la raíz de este proyecto.
2. Levanta el servicio usando el wrapper de Gradle: `./gradlew bootRun`
